# Documentation
service api to get logging by day and monthly from elasticsearch central-logging

Prequirement:
- NodeJS version d18.11.0
- Docker version 20.10.7

Running project via bash script:

* Install packages:

    ```bash
    npm install
    ```

* Running project:

    ```bash
    npm start
    ```
* Open http://your-ip-address:3000 to check it.
